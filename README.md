==========================
【眉毛五種類型（簡化版）】
==========================

1. 劍眉（Sword Brows）
   - 特徵：眉峰高、線條硬、角度明顯
   - 性格傾向：果斷、直接、壓力下仍保持決策能力

2. 一字眉（Straight Brows）
   - 特徵：眉形平直、眉峰不明顯
   - 性格傾向：邏輯型、穩定、做事按程序

3. 柳葉眉（Rounded Brows）
   - 特徵：眉型弧度柔和、眉尾下垂
   - 性格傾向：溫柔、共感強、善於察言觀色

4. 斜飛眉（Arched Brows）
   - 特徵：眉峰明顯向外上揚
   - 性格傾向：強勢、領導力、目標性高

5. 眉短 / 稀疏眉（Sparse Brows）
   - 特徵：眉毛較短、密度稀疏
   - 性格傾向：敏感、缺乏安全感、創造力高
